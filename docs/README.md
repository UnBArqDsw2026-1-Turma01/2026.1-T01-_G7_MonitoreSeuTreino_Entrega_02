## Sobre

O **MonitoreSeuTreino** é um sistema web voltado para praticantes de exercícios físicos que desejam organizar, registrar e acompanhar seus treinos semanais de forma simples e objetiva.

Nesta segunda entrega, a documentação está focada no **Desenho de Software (Modelagem)**, cobrindo os artefatos de modelagem UML estática, dinâmica, organizacional/casos de uso, além das participações e iniciativas extras da equipe.

### Objetivo da Entrega

Consolidar a base de modelagem do projeto, registrar o planejamento do trabalho da equipe e estruturar a documentação da entrega para publicação no GitHub Pages.
